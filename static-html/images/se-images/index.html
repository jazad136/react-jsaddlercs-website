CSCI 4230/4231 - Spring 2021
Course Syllabus

I led section 1 of the departmen's Spring 2021, 16-week Software Engineering Capstone course. This 15-week course was designed to lead seniors through a teamwork intensive semester of understanding how to build enterprise grade software. Twenty-eight students total participated in an intensive software enterprise software development experience. As part of our course learning goals, students were to choose software development processes and employ state-of-the-art tools to develop software in a product-based software engineering learning environment.
