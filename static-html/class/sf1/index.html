SENG 1000 - Fall 2020

SENG 1000 is the first course completed by undergraduates in the ECU Software Engineering program, taught by faculty in the Department of Computer Science. 

We covered in our shortened 8-week term, four key fundamentals of programming in the Java and Python programming language: variable storage and recall, string comparison, control flow and repetition, and classes of variables and methods.

All four topics were taught from the perspective of the Java and Python programming languages. Live lectures were delivered using examples in both Java and Python.  

Assignments were accepted in Java for grading and feedback, and focused on on programming in Java from the Perspective of the following 3 key topics:

  • Finite State Machines
  • |
  • UML Diagrams
  • |
  • Goals, Inputs, and Outputs

Testing covered all seven of these concepts, asking students to (A) recall the facts and (B) demonstrate concepts using graded programming exercises and test questions.

Toward the middle, we covered the Waterfall Lifecycle Model briefly, as an introduction to a simple, corporate development model.
