In Advanced Data Structures and Algorithms, our class covered 4 areas: Sorting, Selection, Memoization and Dynamic Programming, and Graph Search. Students were tasked with reading code implementations as they were demonstrated in class. Demonstrations were conducted in the Python and Java languages. An overview of the C++ language was given three weeks into the course. A tutorial for installing C++ into a home virtual desktop environment was offered. After the overview, after which students could optionally submit code in that language as well as the other languages to receive credit.

Rubric code projects were written in languages students chose for each assignment to potentially accommodate each student's need for assistance working in these environments. In this class we introduced and demonstrated many algorithms, and class demonstration code was completed for 6 of these:
