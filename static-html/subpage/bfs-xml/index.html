Research Project Descriptions

BFS-XML

An XML processing program written in Java, capable of returning elements from an XML document in breadth-first search order. Useful for flattening XML files with a tall hierarchy.

Link to repository: https://github.com/jazad136/bfs-xml

SRCSV

A source code static analysis program written in Java that reads in Java Source code snippets, and returns a table of attributes found at each lineof the snippet. Useful for when statistics are required on Java code, but only part of a compiling code file is available.
